# Blip
Bringing people closer, one message at a time.


** DEPLOYMENT IN PROCESS **


Run Locally
Clone the project

  git clone https://github.com/piyush-eon/mern-chat-app
Go to the project directory

  cd mern-chat-app
Install dependencies

  npm install
  cd frontend/
  npm install
Start the server

  npm run start
Start the Client

  //open now terminal
  cd frontend
  npm start
